The Technique

Demo uses attr() but you don't have to

Implementation Steps:

  1. Grab animation from a prop: <ul style="--_animation: scale-up">
  2. Wire up some keyframes
  3. Attach them to scrollport intersection view()
  4. Run keyframes for element height on entry
